   git clone [https://github.com/your-username/al-aimmah-interactive.git](https://github.com/your-username/al-aimmah-interactive.git)
cd al-aimmah-interactive

 * Install dependencies:
   npm install
# Required libraries: react, framer-motion, lucide-react, tailwindcss

 * Run Locally:
   npm run dev

 * Build for Production:
   npm run build
